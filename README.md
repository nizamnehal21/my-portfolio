# 🌐 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**.
This project showcases personal information, skills, projects, and contact details in a clean and professional layout, designed to represent a developer’s online presence and portfolio.
